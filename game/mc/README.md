# 🎮 Minecraft 商店模拟器

一个基于 Web 的 Minecraft 风格商店经营游戏。

## ✨ 特性

- 🏪 经营你自己的 Minecraft 商店
- 👥 接待各种顾客（Steve、Alex、村民等）
- 💰 赚取金币，扩展业务
- 🎨 Minecraft 风格的像素 UI
- 🔊 音效和动画反馈

## 🚀 快速开始

### 方法 1：本地运行（推荐）
直接双击打开 `index.html` 文件即可开始游戏。

### 方法 2：本地服务器（开发者）
```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000/index.html
```

### 方法 3：部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 上传所有文件
3. 在仓库设置中启用 GitHub Pages
4. 几分钟后就可以在线访问！

详细的部署步骤请查看 [部署指南.md](./部署指南.md)

## 🎯 游戏玩法

1. 点击"⚔️ 开始游戏"进入游戏
2. 铁砧出售基础装备（铁剑、铁镐、铁斧）
3. 箱子出售高级装备（钻石剑、钻石镐、钻石斧、附魔金苹果）
4. 接待顾客，完成订单
5. 赚取金币，升级商店

## 🛠️ 开发

### 本地测试
直接打开 `index.html` 即可，无需服务器。

### 代码结构
- `data.js`：存储游戏数据（装备价格、顾客皮肤等）
- `game.js`：游戏核心逻辑
- `index.html`：UI 结构和样式
